# Hindustan Radiance (HR-News)

**Hindustan Radiance** is a state-of-the-art, premium digital news platform designed to provide real-time updates with a focus on deep insights and unbiased reporting. Built with modern web technologies, it offers a seamless reading experience across all devices.

## ✨ Key Features

- **Live News Feed**: Integration with Google News RSS to provide the latest headlines from across the globe in real-time.
- **Top Headline Spotlight**: A dedicated hero section for the most critical news of the hour.
- **AI-Powered QuickRead™**: Integrated AI summary drawer that provides instant takeaways from complex stories.
- **Dynamic News Cards**: Intelligently designed cards that handle varying headline lengths and provide full excerpts.
- **Smart Image Fallbacks**: A robust system that scans news descriptions for images and provides high-quality, relevant placeholders when source images are missing.
- **Premium Aesthetics**: A custom-designed UI featuring "Radiance Gold" accents, smooth animations with Framer Motion, and a refined typography system.
- **Responsive & Fast**: Fully optimized for mobile, tablet, and desktop viewing using Tailwind CSS v4.

## 🚀 Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS v4 (Vanilla CSS architecture)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Data Layer**: Context API with real-time RSS fetching (rss2json)

## 📖 Usage

- **Browse News**: Scroll through the "Latest Stories" and "Trending Now" sections.
- **Read Full Articles**: Click on the "Read Full Story" link on any news card to open the original source in a new tab.
- **AI Summary**: Click on the AI Feature banner to experience a quick summary of current events.

## 🎨 Design Philosophy

Hindustan Radiance is built on the principle that "The speed of information must never compromise the weight of truth." The design uses **Inter** typography for clarity and a **Midnight & Gold** color palette to evoke trust and authority.
